# Attendance

Каталог `attendance` содержит журнал посещаемости и оценок занятий.

Журнал используется для внутреннего учёта хода курса и анализа динамики группы.
Персональные данные участников не хранятся.

---

## Назначение

Журнал позволяет:
- фиксировать присутствие на занятиях;
- отмечать результаты заданий и работ;
- видеть общую картину по группе;
- отслеживать изменения по ходу курса.

---

## Принцип анонимности

Для участников используются псевдонимы (St01, St02, …).

- реальные имена не хранятся;
- связь псевдонима с человеком существует вне репозитория;
- в GitHub отсутствуют идентифицирующие данные.

---

## Файл attendance.xlsx

Основной файл журнала содержит несколько листов:

- `Students` — список псевдонимов;
- `Attendance` — посещаемость по занятиям;
- `Grades` — оценки по заданиям и работам;
- `Summary` — автоматически рассчитанная сводка.

Все расчёты выполняются внутри файла.

---

## Правила ведения

- посещаемость отмечается сразу после занятия;
- оценки вносятся после проверки работ;
- пустая ячейка означает «нет данных»;
- комментарии используются минимально и нейтрально.

Журнал предназначен для рабочего использования и не публикуется.

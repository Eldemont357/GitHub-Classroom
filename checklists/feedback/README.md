# Feedback

Каталог `feedback` содержит материалы для фиксации обратной связи по занятиям и курсу.

Здесь используются два типа файлов:
- `.md` — шаблоны вопросов (что именно спрашиваем);
- `.xlsx` — агрегированные результаты (что получили и как считаем).

Каталог не предназначен для хранения персональных данных и идентификаторов.
В таблицах фиксируются только агрегированные значения (например, средняя оценка занятия).

---

## Состав каталога

- `lesson-feedback.md` — вопросы после отдельного занятия
- `course-feedback.md` — вопросы по курсу/модулю
- `lesson-feedback.xlsx` — агрегаты по занятиям (L01–L18)
- `course-feedback.xlsx` — итоговые агрегаты по курсу

---

## Принципы

- вопросы короткие и повторяемые;
- предпочтение шкалам и вариантам ответа;
- текстовые комментарии используются минимально;
- результаты фиксируются в обобщённом виде.

# Assessments

В этом разделе находятся задания, которые позволяют
проверить, насколько вы действительно разобрались в теме.

В отличие от `exercises`, здесь:
- меньше подсказок
- больше самостоятельных решений
- важна общая логика, а не отдельные приёмы

Задания из этого раздела помогают понять,
какие части темы усвоены, а к каким стоит вернуться.

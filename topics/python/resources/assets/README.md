# Assets

В этой папке хранятся файлы, которые используются
в учебных материалах (`lessons/*.md`).

Содержимое `assets` предназначено для визуального сопровождения
и дополнительных материалов, а не для самостоятельного изучения.

---

## Что здесь хранится

В папке `assets` размещаются:

- изображения (схемы, диаграммы, иллюстрации)
- скриншоты интерфейсов и примеров
- графики и визуальные пояснения
- другие вспомогательные файлы, на которые есть ссылки из Markdown-уроков

Примеры файлов:
- `.png`, `.jpg`, `.svg`
- `.pdf` (при необходимости)
- другие форматы, используемые как иллюстрации

---

## Как используется содержимое

Файлы из `assets`:
- **не читаются напрямую**
- **не изучаются отдельно**
- используются только через ссылки из `.md` файлов

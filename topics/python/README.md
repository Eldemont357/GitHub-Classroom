# Python

Этот предмет посвящён изучению **основ программирования на Python**  
и использованию Python как инструмента для решения практических задач.

Python в рамках курса используется как:
- язык для освоения базовых концепций программирования
- инструмент для автоматизации и экспериментов
- основа для дальнейших технических предметов

---

## Как работать с этим предметом

Рекомендуемый порядок работы:

1. Прочитайте этот файл целиком
2. Изучайте материалы в папке `lessons`
3. Выполняйте задания из папки `exercises`
4. При наличии — задания из `assessments`
5. После работы с темой заполните `feedback.md`
6. Переходите к следующему предмету или теме

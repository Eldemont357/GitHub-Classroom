# Exercises

В этой папке находятся **практические задания** по теме.

Цель заданий:
- закрепить материал из `lessons`
- научиться применять знания на практике
- развить навык самостоятельного решения задач

Рекомендуется:
- выполнять задания последовательно
- проверять результат и логику решения
- уметь объяснить, как и почему работает ваше решение
- составлять аннотацию к каждому смысловому блоку

Важно не просто получить ответ, а понимать ход решения.
